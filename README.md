# AI Curriculum

A personal space to build AI agent skills from first principles — structured, trackable, tied to real builds.

**10-level core curriculum** covering tool calling through production multi-agent systems.  
**13 capstone projects** spanning portfolio builds through expert-level GenAI lead skills.  
**AWS GenAI parallel track** covering Bedrock and SageMaker for cloud AI roles.

A level is not complete until the build works *and* you can answer the six questions at the end of it.

## How to Use This

1. Start at Level 1. Read the study material. Build the project.
2. Answer the six questions in the level README before moving on.
3. Write a reflection in [JOURNAL.md](JOURNAL.md).
4. Update your XP total and skill status in the tables above.
5. Move to the next level.

Capstone 1 and 3 are the foundation portfolio — both unlock after Level 10. Capstones 4–7 unlock after Level 9 and expand into voice, code execution, evaluation, and multimodal. Capstones 8–10 are Expert-difficulty and require the full 10-level core. Capstones 11–13 are GenAI lead-level: observability, autonomous analysis, and alignment — the skills that separate engineers who build AI from engineers who run it responsibly.

The AWS track is independent and can run alongside the core curriculum from Level 3 onwards.
